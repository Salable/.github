## Salable — Subscription Billing for Developers

[Salable](https://salable.app/) is a developer-first subscription billing platform that lets you launch complex SaaS pricing in minutes, not months. It abstracts away the painful parts — webhook handling, entitlement logic, seat management — so you can focus on shipping your product.

### What we offer

- **Any pricing model** — mix flat-rate, per-seat, usage-based, and one-time charges in a single plan
- **Feature gates out of the box** — check whether a user has access to a feature with a single API call, no feature-flag code needed
- **Team billing** — one subscription covering an entire organisation, B2B-ready from day one
- **Shopping cart & checkout** — bundle products and add-ons into a single Stripe checkout
- **No webhook handlers** — the subscription lifecycle is managed for you
